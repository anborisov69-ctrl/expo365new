# Технический стек EXPO 365

## Основные технологии
- **Next.js 16** - React фреймворк для веб-приложений
- **React 19** - Библиотека для создания пользовательских интерфейсов
- **TypeScript** - Типизированный JavaScript
- **Tailwind CSS** - Утилитарный CSS фреймворк

## Дизайн-система
- **shadcn/ui** - Основной конструктор интерфейса для компонентов
- **Брендовые цвета**:
  - `brand-blue: #0B2B5E` - Основной синий цвет
  - `brand-orange: #F26522` - Акцентный оранжевый цвет

## Архитектура
- **App Router** - Маршрутизация Next.js
- **Модульная структура** - Компоненты организованы в папке `src/components`
- **Модули по доменам** - Логика разделена по папкам в `src/modules` (marketplace, hr-tech, tenders, analytics)

## Установленные компоненты shadcn/ui
- button
- card
- sidebar
- tabs
- table
